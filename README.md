# Neural Decoding Using a Regularized Kalman Filter

This repository presents a machine-learning framework for decoding movement-related kinematic parameters from neural recordings using a Regularized Kalman Filter.

## Overview

Neural decoding is a core component of Brain-Computer Interfaces (BCIs), enabling the prediction of movement intentions and motor behavior from neural activity.

This project investigates the use of local field potentials (LFPs) recorded from the primary somatosensory cortex and applies a Regularized Kalman Filter to estimate continuous kinematic variables.

## Research Objectives

* Decode movement-related kinematic parameters from neural recordings.
* Improve decoding performance through regularization techniques.
* Evaluate neural decoding accuracy for BCI applications.
